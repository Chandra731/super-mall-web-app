# Super Mall Web App

## Overview
This project is a premium-looking, feature-rich, floor-aware e-commerce mall web app using HTML, CSS, JavaScript, and Firebase.

## File Structure
- **public/**: Contains all the frontend files.
  - **css/**: Contains CSS files for styling.
    - `styles.css`: Custom CSS styles.
    - `bootstrap.min.css`: Bootstrap CSS for layout and components.
    - `tailwind.min.css`: Tailwind CSS for utility-first styling.
  - **js/**: Contains JavaScript files.
    - `main.js`: Main JavaScript file for general functionality.
    - `firebase-config.js`: Firebase configuration and initialization.
    - `auth.js`: JavaScript file for authentication logic.
  - **images/**: Contains all image assets.
  - `index.html`: Home Page.
  - `mall.html`: Floor Navigation Page.
  - `shop.html`: Shop Listing Page.
  - `shop-details.html`: Individual Shop Page.
  - `product.html`: Product Page.
  - `cart.html`: Cart Page.
  - `orders.html`: Order Tracking Page.
  - `shop-owner.html`: Shop Owner Dashboard.
  - `admin.html`: Admin Panel.

- **functions/**: Contains Firebase Cloud Functions.
  - `index.js`: Main file for Firebase Functions.
  - `package.json`: Dependencies for Firebase Functions.

- **.firebaserc**: Firebase project configuration.
- **firebase.json**: Firebase configuration for hosting and functions.
- **firestore.indexes.json**: Firestore indexes configuration.
- **firestore.rules**: Firestore security rules.
- **storage.rules**: Firebase Storage security rules.
- **README.md**: Project documentation.
